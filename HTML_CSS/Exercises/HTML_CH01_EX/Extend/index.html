Guideline 1.1 is to provide text alternitives for non-text context. This includes large print, and speech options.

Guideline 1.2 is to provide alternitives to media, such as alternitive media, captions, and discriptions.

Guideline 1.3 is to create content that can be presented in different ways without losing content or structure.

Guideline 1.4 is to make it easier for users to see and hear content to seperate the foreground from the background using audio control, color, contrast, sizes, images, and spacing.
